# Qiskit Developer Certification Study Group 🚀

This repository is a collaborative study hub for preparing for the **Qiskit v2.x Developer Certification Exam**.

## 🎯 Goal

To systematically cover all exam topics through:

* Weekly structured study
* Individual Jupyter notebook contributions
* Group discussions and knowledge consolidation

---

## 📅 Study Plan

We follow an **8-section weekly schedule**, with additional time for revision:

| Week | Topic                 |
| ---- | --------------------- |
| 1    | Quantum Operations    |
| 2    | Visualization         |
| 3    | Quantum Circuits      |
| 4    | Execution             |
| 5    | Sampler Primitive     |
| 6    | Estimator Primitive   |
| 7    | Results & Jobs        |
| 8    | OpenQASM              |
| 9–10 | Revision + Mock Tests |

---

## 📂 Repository Structure

Each section contains:

* `notebooks/` → Individual contributions
* `summary.md` → Consolidated group understanding

---

## 🧑‍💻 Contribution Guidelines

Each week, every member must:

1. Study the assigned section
2. Create a **Jupyter Notebook** including:

   * Key concepts
   * Code examples
   * Edge cases / pitfalls
3. Push to the repository their respective notebook

---

## 🧪 Weekly Meeting Format

* Walkthrough of notebooks
* Discussion of key concepts
* Clarify doubts
* Identify weak areas

---

## 📌 Expectations

* Consistency > perfection
* Clarity over complexity
* Code should run without errors
